## Steps
clone this repo:
1. ```git clone https://github.com/wellpinho/nextjs-clipboard-website.git```
2. ```cd nextjs-clipboard-website/web```
3. ```pnpm i```
4. ```pnpm dev```
5. acccess the project: http://localhost:3000
